# BombaCode Memory

## Mission
BombaCode is a terminal-native coding agent focused on fast, safe, and auditable code changes.

## Current Phase
Phase 1 foundation in progress:
- CLI entry and argument parsing
- Setup wizard and settings persistence
- Ink shell with streaming conversation UI
- Core architecture scaffolding for tools, MCP, memory, and security

## Defaults
- Provider: OpenRouter
- Cost mode: balanced
- Permissions: ask-first for write/bash/network
